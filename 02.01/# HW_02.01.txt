SQL: DDL запросы

  ==========  1  ==========

Спроектируйте базу данных phonebook по описанию в разделе Задание 1 файла # HW_DDL.pdf

Составьте DDL запросы на создание собственно базы данных и таблицы в базе.

    Продумайте, какие типы данных более оптимальны для различных полей таблицы.

Предусмотрите ситуацию, в которой база данных и таблица уже существуют и их необходимо создать повторно.

Все необходимые для создания базы запросы поместите в один .sql файл.

Продумайте необходимое количество таблиц, набор полей для каждой таблицы и связи между ними. Эти решения должны основываться на принципе наименьшего дублирования данных.

Аргументированно выберите тип данных для каждого поля каждой таблицы в базе.

Предусмотрите ситуацию, в которой база данных и таблицы уже существуют и их необходимо создать повторно.

Все необходимые для создания базы запросы поместите в один .sql файл.

  =========================

Запросы для каждого задания помещаются в текущий каталог репозитория в виде отдельного SQL файла с именем, соответствующем номеру задачи.

В сервис Журнал отправляется сообщение о выполнении задания со ссылкой на репозиторий.

Преподаватель пишет комментарии и правки в коде задач и обновляет файлы в удалённом репозитории.
Проверяйте обновления в своём репозитории каждый день.
