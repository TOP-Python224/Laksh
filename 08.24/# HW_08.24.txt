  ==========  2  ==========

Напишите функцию, "обрабатывающую" звуковой файл

С бинарным объектом звукового файла мы работать не будем, поэтому пусть наша функция принимает в качестве аргумента не сам бинарный объект, а строку, содержащую путь к файлу с расширением .wav

Параметры функции: 
 - путь к аудиофайлу: должен заканчиваться именем файла с расширением .wav
 - аудиоформат: целое число или строка в диапазоне от 0 до 9999
 - количество каналов: целое число или строка в диапазоне от 1 до 10
 - частота дискретизации (сэмплирования): целое число или строка из списка допустимых значений (8000, 11025, 16000, 22050, 32000, 44100, 48000, 88200, 96000, 176400, 192000, 352800, 384000)
 - разрядность (битовая глубина): целое число или строка из списка допустимых значений (8, 16, 24, 32)
 
Самостоятельно определите, какие параметры должны быть строго позиционными, какие позиционно-ключевыми, а какие строго ключевыми. 
Аргументируйте в комментариях

Функция должна проверять корректность переданных аргументов и печатать результат проверки. Протестируйте как с корректными данными, так и с некорректными.



  ==========  3  ==========

Напишите функцию, которая удаляет n минимальных и n максимальных значений из списка чисел

Первым аргументом функция принимает список (list)
Вторым – число n
Третьим – логическое значение True или False

Самостоятельно определите, какие параметры должны быть строго позиционными, какие позиционно-ключевыми, а какие строго ключевыми. 
Аргументируйте в комментариях

В зависимости от значения третьего аргумента функция либо редактирует сам переданный список (True) и возвращает None, либо возвращает отредактированную копию списка (False)

Подсказка: можете использовать код для задачи 112 из задания 03.24

Протестировать со случайными числами: используйте функции модуля random или заранее сгенерированные данные (generatedata.com)



  ==========  4  ==========
  
Модифицируйте функцию из задачи 5 задания 08.22 таким образом, чтобы она могла кроме итерируемых объектов альтернативно принимать в качестве аргументов произвольное количество int и/или float объектов

Таким образом, функция принимает либо один итерируемый объект, либо произвольное количество объектов int и/или float – по аналогии с встроенными функциями min() и max()

Примеры вызова функции:
    averages([1, 2, 3, 4.23])   # корректный
    averages('0 2 0.12 1')      # корректный
    averages(-1, 1.5, -2, 2.5)  # корректный
    averages('0b010 0b110')     # некорректный
    averages(1, 2, '3 4')       # некорректный
    averages([1, 2], '3 4')     # некорректный
