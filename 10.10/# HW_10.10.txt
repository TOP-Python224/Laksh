Проект 1. Интегрированные среды разработки: PyCharm. Создание проекта

  ==========  1  ==========

Скачать и установить приложение PyCharm Community Edition:

    https://www.jetbrains.com/pycharm/download/


В настройках PyCharm (Ctrl+Alt+S) подключить свой аккаунт GitHub (Version Control -> GitHub -> + -> Log In with Token... -> Generate...)


Создать заготовку для проекта:

    1. В PyCharm создайте новый проект. В окне создания проекта:
        1.1. Выберите в локальной файловой системе каталог для расположения проекта
        1.2. Допишите к пути расположения проекта имя нового вложенного каталога — это имя станет именем проекта по умолчанию
        1.3. Выберите создание нового виртуального окружения VirtualEnv
        1.4. Запустите создание нового проекта
    
    2. В каталог проекта добавьте файл .gitignore (образец файла в проекте-референсе)
    
    3. В GitHub Desktop создайте новый локальный репозиторий в каталоге проекта:
        3.1. File -> New repository...
        3.2. В поле Name укажите имя каталога с проектом (из п.1.2)
        3.3. В поле Local path укажите путь к каталогу, содержащему каталог проекта (из п.1.1)
        3.4. Create repository
    
    4. Отправьте локальный репозиторий на удалённый сервер GitHub:
        4.1. Publish (на панели инструментов)
        4.2. В поле Name введите имя по шаблону: <имя_репозитория_с_ДЗ>_project1
        4.3. Снимите галочку Keep this code private
        4.4. В выпадающем списке выберите организацию TOP-Python224
        4.5. Publish



  =========================

Ссылка на удалённый репозиторий проекта помещается в текущий каталог репозитория в виде TXT файла с именем, соответствующем номеру задачи.

В сервис Журнал отправляется сообщение о выполнении задания со ссылкой на репозиторий.
